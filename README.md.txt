# 📊 SQL Exploratory Data Analysis Project

## 📝 Overview
This project performs **Exploratory Data Analysis (EDA)** using raw SQL queries on a relational database. It walks through common data analysis operations such as data inspection, aggregation, joining, and trend analysis using SQL only.

---

## 🧰 Tools Used
- SQL (MySQL)
- Any SQL GUI Tool (like MySQL Workbench, DBeaver, etc.)
